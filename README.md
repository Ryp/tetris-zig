# Tetris

## Building

This should get you going after cloning the repo:
```sh
$ zig build run
```

## Controls

| Action                          | Key                             |
|---------------------------------|---------------------------------|
| Push piece down                 |  Down arrow                     |
| Push piece sideways             |  Left/Right arrow               |
| Rotate piece counter-clockwise  |  W                              |
| Rotate piece clockwise          |  E                              |
